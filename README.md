# Number to Word Converter

## Exercise description

Write some code that will accept an amount and convert it to the
appropriate string representation.
 
_Example:
Convert 2523.04
to "Two thousand five hundred twenty-three and 04/100
dollars"_

## How to run

* Save the project to your computer
* Open **NumberConverter.html**
* Type an amount into the input box to see the result in real-time

## How to test

* Test cases run automatically after the page loads
* To see if a test failed, open the console and see if there are any errors logged
